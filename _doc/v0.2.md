# v0.2

## 版本資訊
- 發布日期：2026-06-01
- 版本狀態：開發中

## 更新內容

### 新增功能

#### 五子棋 (Gomoku) AI 大幅強化
- **Minimax + Alpha-Beta Pruning 演算法**：深度 1-4 層
- **棋型評分系統**：
  - 五連（FIVE）：1,000,000 分
  - 活四（LIVE_FOUR）：100,000 分
  - 衝四（RUSH_FOUR）：10,000 分
  - 活三（LIVE_THREE）：10,000 分
  - 眠三（SLEEP_THREE）：1,000 分
  - 活二（LIVE_TWO）：500 分
  - 眠二（SLEEP_TWO）：100 分
- **難度對應搜尋深度**：
  - Level 1-2：深度 1 + 高隨機
  - Level 3：深度 2 + 低隨機
  - Level 4：深度 3（強）
  - Level 5：深度 4（最強）
- **快速必殺棋檢測**：優先處理獲勝/防守棋步

#### 中國象棋 (Chinese Chess) AI 強化
- **保護將的邏輯**：
  - 移動後將被吃 → 跳過此移動
  - 移動後將被將軍 → 大幅扣分（-2000）
  - 解除將軍 → 獎勵 +800
  - 堵塞攻擊路線 → 獎勵 +500
- **九宮防守評估**：士/象在九宮內加分
- **將軍威脅評分**：將對方將軍 +500 分
- **難度對應搜尋深度**：
  - Level 1：深度 2
  - Level 2-3：深度 3
  - Level 4-5：深度 4
- **隨機因素**：低難度加入隨機性降低 AI 強度

#### 數獨 (Sudoku) 操作優化
- **彈出式數字選擇器**：
  - 點擊空格後跳出 1-9 數字按鈕
  - 智能定位（避免超出螢幕）
  - 支援清除功能
- **移除側邊數字鍵盤**：減少視覺干擾
- **操作提示**：提醒使用者點擊空格

### 技術改進

#### 五子棋 AI 架構
```javascript
// 棋型分析函數
analyzeLine(board, row, col, dr, dc, player)
  → { count, block, empty }

// Minimax + Alpha-Beta
minimax(board, depth, alpha, beta, isMaximizing)
  → 評估分數

// 候選位置優化：只考慮有棋子周圍 2 格範圍
getCandidates(board) → 限制搜尋範圍提升效能
```

#### 中國象棋 AI 架構
```javascript
// 將的位置追蹤
findKing(color, board) → { row, col }

// 位置是否被攻擊
isSquareAttacked(row, col, byColor, board) → boolean

// 將是否被將軍
isKingInCheck(color, board) → boolean

// 移動安全性評估
if (!findKing('black', newBoard)) continue;  // 跳過自殺棋
if (isKingInCheck('black', newBoard)) score -= 2000;  // 將軍懲罰
```

# v0.1

## 版本資訊
- 發布日期：2026-06-01
- 版本狀態：開發中

## 更新內容

### 新增功能

#### 遊戲選單架構
- 建立遊戲選單首頁 (GameMenu)
- 使用 React Router 實現路由
- Lazy Loading 延遲載入各遊戲模組
- iPad 響應式設計支援

#### 西洋棋 (Chess)
- 移植自原有 chess-game 專案
- AI 對戰功能
- 難度調整 (1-5 + hope/desperate)
- 棋路建議功能（三色標記）
- iPad 優化顯示

#### 中國象棋 (Chinese Chess)
- 完整棋規實作（帥/仕/相/馬/車/炮/兵）
- AI 對戰功能
- 難度調整 (1-5)
- 九宮格與河界限制
- 棋子移動合法性驗證

#### 數獨 (Sudoku)
- 自動生成謎題與解答
- 四種難度等級（簡單/中等/困難/專家）
- 提示功能（3 次）
- 錯誤檢查標記
- 計時功能

#### 五子棋 (Gomoku)
- 三種棋盤大小 (13×13 / 15×15 / 19×19)
- AI 對戰功能
- 難度調整 (1-5)
- 勝利連線顯示
- 最後落子標記

### 變更
- 專案名稱從 `react-chess-game` 改為 `game-hub`
- 新增 react-router-dom 依賴
- 重構為模組化架構
