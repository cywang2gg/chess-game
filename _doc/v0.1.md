# v0.1

## 版本資訊
- 發布日期：2026-06-01
- 版本狀態：開發中

## 更新內容

### 新增功能

#### 遊戲選單架構
- 建立遊戲選單首頁 (GameMenu)
- 使用 React Router 實現路由
- Lazy Loading 延遲載入各遊戲模組
- iPad 響應式設計支援

#### 西洋棋 (Chess)
- 移植自原有 chess-game 專案
- AI 對戰功能
- 難度調整 (1-5 + hope/desperate)
- 棋路建議功能（三色標記）
- iPad 優化顯示

#### 中國象棋 (Chinese Chess)
- 完整棋規實作（帥/仕/相/馬/車/炮/兵）
- AI 對戰功能
- 難度調整 (1-5)
- 九宮格與河界限制
- 棋子移動合法性驗證

#### 數獨 (Sudoku)
- 自動生成謎題與解答
- 四種難度等級（簡單/中等/困難/專家）
- 提示功能（3 次）
- 錯誤檢查標記
- 計時功能

#### 五子棋 (Gomoku)
- 三種棋盤大小 (13×13 / 15×15 / 19×19)
- AI 對戰功能
- 難度調整 (1-5)
- 勝利連線顯示
- 最後落子標記

### 變更
- 專案名稱從 `react-chess-game` 改為 `game-hub`
- 新增 react-router-dom 依賴
- 重構為模組化架構

### 技術架構

```
src/
├── App.js                 # 主應用入口 + 路由配置
├── components/
│   └── GameMenu.js        # 遊戲選單頁面
├── games/
│   ├── ChessGame/         # 西洋棋模組
│   ├── ChineseChess/      # 中國象棋模組
│   ├── Sudoku/            # 數獨模組
│   └── Gomoku/            # 五子棋模組
└── index.js               # React 入口
```
